Конфигурация - 
    Видео разогрев

    Текст и фото для случайных постов
    Текст для дневного разогрева (за день до мероприятия)
    Текст для 2х часового разогрев
    Текст для 1 часового разогрев
    Текст для старта
    
    Кнопки

DB формат - 
    users - 
        id - int(64) хранит номер пользователя в базе
        nickname - varchar(255) хранит никнейм

        name - varchar(255) хранит имя (если есть в тг)
        lastName - varchar(255) хранит фамилию (если есть в тг)
        email - varchar(255) хранит email (если его дали)
        stadia - int(10) хранит текущий этап пользователя (Получать уведомления, получать разогревы, получил разогрев т.д.)
        dataReg - datetime(format) хранит дату регистрации
    

Сценарий - 
    standart - 
        Пользователь - нажимает кнопку старт

        Bot - 
            заносит в базу данных
            Отправляет разогрев (Видео заготовленные зарание)
            После каждые 4часа отправляет случайный пост
            На момент разогрева отправляет тексты (2х, 1 часового разогрева)
        
        Идеи - 
            Можно запросить почту, чтобы мы могли оповестить там
            Можно подключить API чтобы оповестить по sms
